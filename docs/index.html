Grafana Assistant Workshop

A hands-on workshop teaching you to query, investigate, and build dashboards using Grafana's AI assistant.

Instructor Slides
Lab 00 ~15 min
Getting Started

Log in, create a personal folder, orient to the environment, meet Grafana Assistant.
Lab 01 ~30 min
Memories & Infra Scan

Run a discovery scan, create custom rules, see how context improves Assistant's answers.
Lab 02 ~20 min
MCP Servers

Explore what tools Assistant has access to and how MCP extends its capabilities.
Lab 03 ~40 min
Queries & Dashboards

Query logs, metrics, and traces in plain English. Explain panels. Build a dashboard.
Lab 04 ~60 min
Investigation

Investigate a real incident end-to-end. Launch a multi-agent swarm investigation.
